About

Northwestern EECS 395: Engineering Systems Design I

This site is designed to display an image uploaded by a webcam created during the first iteration of Engineering Systems Design. The webcam is an embedded system built over the course of a quarter. It refreshes approximately every 4 seconds and uploads 640px x 480px images to the webcam.html page.

Per the class description:

This fast-paced course will teach a student how to go from a project idea to a fully functional prototype implementation. This involves a printed circuit design using PCB CAD software, surface mount soldering, MCU programming, CAD design for 3D printing, Android programming, and web design
